# Описание процесса тестирования оборудования

Для выпуска продукции фирма установила оборудование. Каждая единица оборудования относится к одному из типов оборудования.

В целях обеспечения безопасности работы оборудование должно регулярно по графику проходить тестирование.

## Характеристики оборудования

Для каждой единицы оборудования хранится:
- **Тип оборудования**
- **Уникальный код**
- **Название**
- **Производитель**
- **Дата проведения последнего тестирования**
- **Статус оборудования**:
  - `1` — оборудование успешно прошло плановые тесты
  - `2` или `3` — тесты пройдены с замечаниями

## Сотрудники

Тестирование проводят сотрудники фирмы, о которых известны:
- Ф.И.О.
- Дата рождения
- Шифр отдела, в котором работает сотрудник

## Протокол тестирования

В протокол тестирования обязательно заносится:
- Дата проведения тестирования
- Код сотрудника, проводившего тестирование
- Код единицы тестировавшегося оборудования
- Статус, присвоенный оборудованию по результатам тестирования

## Тесты

К каждому протоколу прилагается список тестов с указанием:
- **Названия теста**
- **Кода его завершения**:
  - `1` — успешно пройден
  - `0` — неуспешно

Сотрудник, проводящий тестирование, может выбрать необходимое количество тестов для конкретной единицы оборудования из списка возможных тестов для данного типа оборудования.
